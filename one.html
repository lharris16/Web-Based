1.4

A fully qualified domain name is the host name and all the domain names together. The domain name after the host is which domain the host is a part. The second domain name is which domain the first
is a part. The last domain is the type of organization in which the host exists in.

must be converted to IP addresses so the message can get transmitted over the Internet to the destination.
